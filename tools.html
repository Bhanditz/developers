Developers are welcome to get involved by contributing code, either to existing tools or proposed future projects.

But it's not the only way to get involved in making W3C developer tools better. Testing, bug reports, suggestions, or help in creating good documentation are equally important! Most project will have a Feedback page, and you can report bugs, test cases and patches.
