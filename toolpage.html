Unicorn

the unified validator

Unicorn is W3C's unified validator, which helps people improve the quality of their Web pages by performing a variety of checks. Unicorn gathers the results of the popular HTML and CSS validators, as well as other useful services.

You can tell Unicorn what to validate in any of the following ways:

  • HTML5 (and older) development
  • CSS3 (and older) development
  • Mobile content development
  • RSS integration
  • Check avaibility of Links

Help us to improve Unicorn in making donations

For almost 14 years, the W3C has been developing and hosting Web Tools used every day by millions of Web developers and Web designers. All these tools are free, Open-Source and provides the web with a fair, reliable help in building web content that works now and will work in the future.

Good tools cost a lot to develop, support, host and maintain: with your help, we can make the validators better, faster, friendlier, and maintain a great, free service for all of the Web.

Donations to this program will be used exclusively for the maintenance, hosting and development of our free validation tools and other similar open source projects.
